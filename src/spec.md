# Valentine's Day Interactive Website

## Overview
A single-page Valentine's Day website that presents a romantic proposal to Nitin with interactive elements and animated decorations.

## Core Features

### Main Message Display
- Display the message "Dear Nitin, will you be my Valentine?" prominently on the page

### Interactive Buttons
- Two buttons: "Yes" and "No"
- **Yes Button**: When clicked, displays a cheerful confirmation message "Yay! I knew it ❤️"
- **No Button**: Cannot be successfully clicked - each attempt shows playful messages in sequence or randomly:
  - "Are you sure?"
  - "Don't you love me?"
  - "Please baby"
  - "No, this is not happening"
  - "This is not happening"
  - "You have to say yes"
  - "Come on, it's Valentine's!"

### Visual Design
- Valentine's color palette using pinks, reds, and whites
- Centered, responsive layout that works on all screen sizes
- Animated floating balloons in the background for romantic atmosphere
- Cute, romantic styling throughout

### Technical Requirements
- Single standalone HTML file containing all HTML, CSS, and JavaScript
- No external dependencies or files
- Fully self-contained and functional
- English language content

## User Experience
The website creates a playful, romantic interaction where the user is gently prevented from declining the Valentine's proposal through humorous messages that rotate randomly or sequentially, while celebrating an affirmative response with joy. The "No" button maintains its movement behavior to make the interaction more fun and unpredictable.
